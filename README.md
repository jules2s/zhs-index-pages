# zhs-index-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面均为静态内容，不依赖后端服务，可独立在浏览器中打开或部署至任意静态托管平台。

## 项目简介

`zhs-index-pages` 是一个以 HTML 页面为核心的归档仓库。仓库内每个页面均为独立的文件，彼此之间没有功能关联，仅作为资料存档或索引展示使用。适合存放各类信息页面、导航页、公告页等静态内容。

## 目录说明

```
zhs-index-pages/
├── index.html          # 仓库的首页或总索引
├── pages/              # 归档页面存放目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 公共资源目录（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- **index.html**：仓库的入口页面，通常用于展示页面列表或导航。
- **pages/**：存放所有归档的 HTML 页面，文件命名建议保持有序和清晰。
- **assets/**：存放页面共用的样式、脚本或图片等静态资源。

## 页面归档说明

- 每个页面均为独立、完整的 HTML 文档。
- 页面内容以信息展示为主，不包含动态交互或外部请求。
- 归档页面可随时添加、修改或移除，保持仓库整洁即可。
- 建议为每个页面添加简短的说明注释，便于后续维护。
